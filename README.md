# Project #1 - Game

Game ini dibuat menggunakan HTML, CSS, JavaScript dan Python.

## Cara Menjalankan dari Powershell:
1. Pastikan Anda memiliki Node.js terinstal.
2. Jalankan `npm install` untuk menginstal dependency.
3. Buka `index.html` di browser atau jalankan melalui Electron jika tersedia.

## Cara Menginstall lewat Release:
1. Pastikan Anda telah mendownload installer game dari Release.
2. Run Installer dan ikuti prompt yang muncul di pop-up window.
3. Setelah meng-install, anda dapat memainkan game tersebut. enjoy!

## Struktur Proyek
- `index.html`: Struktur utama game.
- `script.js`: Game Logic.
- `style.css`: UI/UX.
- `ai_engine.py`: AI Engine.
